# ASO
A clothing brand website celebrating West African and Cultural textile heritage - blending traditional patterns with modern design.
## About
ASO is a cloth brand gotten from passion and inspirations rooted in cultural desires and traditional textile, reimagined for a contemporary audience. This site showcases the brand"s identity, visual aesthetic, and collections.
## Design
The site draws on adire dyeing traditions, built around a palatte of deep indigo, gold and earth tones to reflect the brand's cultural roots.
## Features
- Responsive layout for mobile and desktop
- Brand story and heritage section
- Visual showcase of collections/designs
- Contact/inquiry section
## Tech Stack
- HTML5
- CSS3
- JavaScript
